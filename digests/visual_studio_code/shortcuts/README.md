# 비주얼 스튜디오 코드 단축키

## 편집

* 한줄지우기 Ctrl + Shift + K
* 단어 바꾸기 Ctrl + H

## 이동

* 첫번째 줄 Ctrl + Home
* 마지막 줄 Ctrl + End

## 탭

* 탭 나누기 Ctrl + \
* 탭 닫기 Ctrl + F4
* 탭 리스트 Ctrl + Tab
* 탭 좌 이동 Ctrl + Page Up
* 탭 우 이동 Ctrl + Page Down

## 마크다운

* 미리보기 탭 열기 Ctrl + Shift + V
* 미리보기 옆 열기 Ctrl + K, V (vim 플러그인과 충돌)
