# 비주얼 스튜디오 코드 플러그인: Vim

## 설정

### 속성

* `"vim.useCtrlKeys"`: 컨트롤 기반 쇼컷 지원
* `"vim.useSystemClipboard`: 시스템 클립보드 연동 
* `"editor.cursorStyle"`: 플러그인 비활성화시 커서 모양 설정

### 예시

    {
        "vim.useCtrlKeys": "true",
        "vim.useSystemClipboard": "true",
        "editor.cursorStyle": "underline"
    }
